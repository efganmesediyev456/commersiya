## Getting Started with Development
This is a development setup guide for this project. Please follow the instructions to setup
project locally.

### ü§ö **Requirements**
- PHP 7.1.2
- Mysql 5.6

### ‚è≥ **Running locally**
- Pull the repo
    ```bash
    cd avirtv
    ```
- Install requirements
    ```bash
    composer install
    ```
- Change .env db config with your db
    ```bash
    DB_HOST=127.0.0.1
    DB_PORT=3306
    DB_DATABASE=avirtv
    DB_USERNAME=root
    ```
- Migrate
    ```bash
    php artisan migrate
    ```
  
- To run on local
    ```bash
    php artisan serve
    ```

    Application will run on [127.0.0.1:8000](http://127.0.0.1:8000)
